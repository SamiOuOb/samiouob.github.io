[Raspberry Pi 3 B+] 拿到樹莓派後我該做什麼？

Posted on 2019-05-04


某天，我從即將飛去德國的學長手中接下了數台樹莓派，
在這之前我完全沒有使用過樹莓派或是其他開發版的經驗，
所以這個看起來有夠奇怪的小盒子到底該怎麼用，
我根本一點頭緒都沒有。


不過既然天上掉下來一個新玩具，
沒有道理不好好的、徹底的玩不學習啊，
所以我就開始研究這個小盒子到底能幹嘛。

硬體規格

SoC:        Broadcom BCM2387B0 chipset
CPU:        1.4GHz Quad-Core ARM Cortex-A53 (64Bit)
記憶體:        1GB LPDDR2（和 GPU 共享）
視訊輸出:    Composite RCA; HDMI
音訊輸出:    3.5 mm jack; HDMI(1.3 & 1.4)
儲存:        microSD
USB:         USB 2.0 x 4
工作電流:    459mA(2.295W) when idle/1.13A(5.661W) maximum under stress
尺寸:        85mm x 56mm x 17mm
重量:        50g

我先上網看了一下他的規格，
Raspberry Pi 3 B+的CPU是四顆A53構成，
基本上就是一半的Snapdragon S625，
也就是一半ASUS Zenfone3的CPU效能，
算是相當不錯(S625以省電著稱)，

不過只有1GB的DDR2就讓我稍微感到毛毛的了XD。

OS安裝

Ubuntu或是Window是將ISO檔燒到USB/CD裡做成開機碟，
再用開機碟開機進行安裝；
但樹苺派的OS安裝並非如此，
而是將OS映像檔(.img)用程式燒進microSD卡裡面，
樹莓派會直接用microSD開機。

下載映像檔

到官網下載Raspbian Stretch，
下載頁面中會看到三種選擇，
分別是：

  • Raspbian Stretch with desktop and recommended software
    (含桌面程式與預裝建議軟體)
  • Raspbian Stretch with desktop
    (含桌面程式)
  • Raspbian Stretch Lite
    (純文字介面)

選擇你所需要的版本，
用力按下Download ZIP就對了。

將映像檔燒進micorSD

以Windows為例，我們需要win32diskimager這個程式。

點Download後會倒數幾秒鐘，等倒數完就會自動開始下載。


下載完後把它安裝，
然後先去把剛剛從官網下載下來的映像檔ZIP解壓縮。

打開win32diskimager後會是這個頁面，點紅色框框的圖示：


選擇解壓縮出來的映像檔。


點選**資料到「裝置**，然後選Yes。


綠色進度條如果有開始動代表成功開始燒錄惹。


過程不會太久，應該10分鐘內就可以完成OuOb。

至此OS安裝的部分已全數完成，
接下來把SD卡插入樹莓派就能開機進入系統摟，

我好興奮！我好興奮啊！
