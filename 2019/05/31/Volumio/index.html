[樹莓派 Raspberry Pi] 用 樹莓派 + volumio 製作HIFI播放器吧!

Posted on 2019-05-31 | 閱讀次數

前言

之前提到，我從去德國交換的學長手中接下了數台樹莓派，
不過暫時沒有需要用到他們的地方，
後來想想與其放在那邊生灰塵，
不如拿來組個PiFi DAC來推我的耳罩看看(?

其實我本來就是個喜歡玩耳機的肥宅，
不過手邊只有一個便宜耳擴，
卻沒有堪用的DAP，
這 怎 麼 能 忍 呢 !
所以我就決定動手做了。

材料

Raspberry Pi 3 B+ x 1
記憶卡 x 1
USB3.0 隨身碟 x 1

當然還有這次的主角
PiFi DAC + 2.0 x 1

OS選擇

我選擇使用比較多人用的 volumio 作為OS，
好處是使用簡單、設定齊全，
也支援大部分的樹莓派DAC擴充版，十分方便。

除此之外就是，volumio有很多外掛可以用，
有了外掛他甚至還能串流播放Youtube、Spotify的音樂。

Volumio 安裝

下載映像檔

到官網 下載 volumio，
點選下載頁面的 Download 就會開始下載。

下載完後將ZIP檔解壓縮。

將映像檔燒進micorSD

volumio 也是利用 win32diskimager 這個程式進行燒錄，
操作過程跟在[Raspberry Pi 3 B+] 拿到樹莓派後我該做什麼？中講過的一樣，
安裝完成後把記憶卡塞進樹莓派就能開機摟，
第一次開機由於會進行一些程式的設定跟初始化，
所以會花比較多時間，當可以搜尋到名為volumio的WIFI時，
代表已經開機完成了。

Volumio 初次設定

第一次開啟 Volumio 作業系統的時候，
必須進行一些設定，
不過相當簡單，並不需要打程式碼，
且介面也非常易懂。

第一步 如何進入 Volumio 設定頁面?

首先用手機或電腦搜尋一個叫作 Volumio 的 WIFI熱點，
(預設密碼為volumio2，之後可作更改)
連上之後在網頁輸入網址 volumio.local/。

第二步 初始化設定

  1. 選擇語言 有繁體中文可供選擇

  2. 設定裝置名稱

    這會影響日後用手機或電腦進入控制頁面所要打的網址，
    像我設定的名字是 SamiMusic，
    那以後就必須輸入 samimusic.local/ 來進入我的控制頁面，
    當然，也可以用預設的 volumio就好。

  3. 選擇你的DAC型號

    若你沒有 I2S DAC，也可選擇內建的耳機孔輸出

  4. 連WIFI

  5. 選擇音樂資料庫

  6. 自由贊助
    如果不想贊助，直接按下一步即可。

到這邊初次設定已經完成，可以成功進入播放頁面。
